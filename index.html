🔑 Google Calendar API セットアップ
1
Google Cloud Console console.cloud.google.com を開く
2
新規プロジェクト作成 「新しいプロジェクト」→ プロジェクト名を入力 → 作成
3
Calendar API 有効化 「APIとサービス」→「ライブラリ」→「Google Calendar API」→ 有効化
4
OAuthクライアントID作成 「認証情報」→「+認証情報を作成」→「OAuthクライアントID」→「ウェブアプリ」
5
オリジンを追加 「承認済みのJavaScriptオリジン」に http://localhost と http://127.0.0.1 を追加
6
クライアントIDをコピー 作成されたクライアントID（〜.apps.googleusercontent.com）をコピー
📅
予定を追加
🔍
予定を確認
